# GeneralTable

The GeneralTable component facilitates the presentation of data in a tabular format, incorporating pagination functionality.
Additionally, it provides comprehensive insights into the total number of rows. It includes features to indicate
loading states and handle potential errors during data retrieval. Check it out by changing "loading" to "true" and write error message in "error".
